Marks Evolution System
📌 Overview

The Marks Evolution System is a Python-based desktop application developed using Tkinter and MySQL.
It helps in storing, calculating, and retrieving student semester results in a simple and efficient way.

🎯 Objectives
Store student academic details securely
Calculate total marks, average, and result classification automatically
Retrieve student results using Hall Ticket Number
Provide a user-friendly graphical interface
🛠 Technologies Used
Python
Tkinter (GUI)
MySQL
mysql-connector-python
✨ Features
Student marks entry system
Automatic calculation of total, average, and result
MySQL database integration
Result retrieval using HT number
Simple and beginner-friendly interface
🧑‍💻 Modules
1️⃣ Student Data Entry Module
Enter HT number, name, and subject marks
Stores data into MySQL database
2️⃣ Result Display Module
Fetches and displays student results using HT number
🗄 Database Details

Table Name: result
Fields:
htno
name
s1
s2
s3

avg
grade
▶️ How to Run

Run the Python files
